## 安装git、nodejs、npm

安装后查看以下命令确认：

    git -v
    node -v
    npm -v

因为“墙”，你可能需要设置代理：

    git config --global http.proxy http://172.19.1.2:9217
    git config --global https.proxy http://172.19.1.2:9217
    git config --global http.sslverify false

    npm config set proxy=http://172.19.1.2:9217
